## Ветки. Локальная работа с ветками в git.
- Для создания ветки
    - _git branch "<имя ветки>"_
- Для перехода в созданную ветку git 
    - _git checkout "<имя ветки>"_
- Или можно совместить:
    - _git checkout -b "<имя ветки>"_
- Для возврата в основную ветку
    - _git checkout master_
- Посмотреть все ветки (в т.ч. активную):
    - _git branch_
